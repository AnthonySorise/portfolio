About Anthony

I love creating things

I'm driven by the enjoyment I find with the creative process. I've explored this passion in many ways, including the cherished experiences of working in audio production on some of the biggest AAA games of the past ten years. As a result of my proximity to the game industry, and my interest in game design - I developed a strong curiosity for computer programming. This curiosity turned into fascination as I taught myself C# and Unity. I discovered that my enthusiasm for creating things translates very well to development. Wanting to deepen my understanding, I recently completed an accelerated web development program where I enjoyed over 400 hours of training using a variety of technologies. I'm eagerly seeking new opportunities to apply and build upon what I've learned so far.

TheCubeTube.com

A website created in a team of six developers that allows users to follow and organize YouTube channels into categories, and provides more options for watching them - including the ability to make an on-the-fly playlist. I created the original idea, and served as project lead. Many of my duties were related to creating nested AJAX calls to retrieve and insert data to the back-end, dynamically loading information on the page using jQuery, and tracking the project's progress using Meistertask.

Live Site

Github

Fallout Scavenger

A Fallout themed match game with additional mechanics including items, health points, and damage resistance. I utilized HTML5 to create the skeleton, CSS and Bootstrap to apply styling, and Javascript and JQuery to create the game logic.

Live Site

Github

Tic Tac Duck Hunt

A Duck Hunt themed Tic Tac Toe “shooting gallery” hybrid created in a team of three developers. My main duties focused on the “shooting gallery” part of the game, using Javascript and JQuery to dynamically display elements for the user to shoot.

Live Site

Github

Pokemon Speed Battle

A Pokemon themed, time-based two-player game created in a team of four developers that utilizes a Pokemon card API, YouTube's API, and web scraping of a Pokemon wiki to provide a unique experience every time it's played. My main contribution was using Javascript to develop the game logic and handle user input.
