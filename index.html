About Anthony

I love creating things

From a young age I've been driven by the enjoyment I find with the creative process. I've explored this passion in many ways, including the cherished experiences of working in audio production on some of the biggest video games of the past ten years including XCOM II, God of War: Ascension, and Call of Duty : Black Ops II. As a result of my proximity to the video game industry, and my interest in game design - I developed a strong curiosity for computer programming. This curiosity turned into fascination as I taught myself C# and Unity. I discovered that my enthusiasm for creating things and problem solving translates very well to development.

With a strong desire to deepen my understanding, I recently completed the LearningFuze accelerated web development program where I enjoyed over 800 hours of training using a variety of technologies. I'm eagerly seeking new opportunities to apply and build upon what I've learned so far.

TheCubeTube.com

With an anonymous unique user web address, users can follow and organize YouTube channels into separate categories. Videos can be loaded from specified channels, and queued in a custom watch order. On a team of six developers, I served as lead throughout the software development life-cycle. Many of my duties were related to using YouTube's API, creating nested AJAX calls to retrieve and insert data to the back end, and using jQuery to dynamically display information to build an effective user interface with Bootstrap.

Fallout Scavenger

A Fallout themed match game with additional mechanics including items, health points, and damage resistance. To enhance the user experience and communicate the game mechanics to the user - I developed a message log system with jQuery. I also used jQuery and CSS to create an animated user interface to capture the attention of the user when the values for health and damage resistance change.

Live Site

Tic Tac Duck Hunt

A Duck Hunt themed Tic-Tac-Toe “shooting gallery” hybrid created in a team of three developers as part of a 48 hour hackathon. I developed the “shooting gallery” part of the game, using Javascript and jQuery to dynamically display elements for the user to shoot. I also optimized audio implementation with Javascript, and edited art assets using Photoshop. Object-Oriented Programming was utilized to separate the “shooting gallery” layer of the game from the Tic-Tac-Toe layer, allowing them to efficiently interact with each other. Keyframe animation in CSS is used to animate the duck and dog sprites.

Live Site

Github

Pokemon Speed Battle

A Pokemon themed, two-player speed sequence game created in a team of four developers as part of a 48 hour hackathon. It utilizes a Pokemon card API, YouTube's API, and web scraping of a Pokemon wiki to provide a unique experience every time it's played. My main contribution was using Javascript to develop the game logic and handle user input. An option for onscreen inputs in lieu of keyboard inputs was developed for compatibility with tablets. This project was created using a Model-View-Controller architecture.
